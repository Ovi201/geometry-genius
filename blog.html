What Is DOM?

DOM is document object model. which is majorly used for modifying or interacting with objects present on a page using Javascript or Jquery. As the name suggests this can be used to modify elements properties like styling, content. It also can be used to retrieve content or the values of form elements.

How do I access DOM elements?

1.The getElementById Method This method name is self-descriptive about its action. It fetches the DOM element with the help of the id attribute.
2.The getElementsByClassName method This method fetches elements with the class attribute. This method returns a NodeList of all the elements that has the specified class attribute.
3.The getElementsByName method This method fetches elements with the name attribute and returns a NodeList. It fetches all the elements that have the specified name attribute.
4.The getElementsByTagName method This method fetches elements with the specified tag name. This method returns a NodeList of all the elements that has the specified tag name.
5.The querySelector method This method uses CSS selectors to access a DOM element and returns the first matched element.

What are Callback Functions and How Do They Work? ?

A callback function in javascript is a function that is passed as an argument in another function.

What is meant by event bubbling?

It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.
